MCP 연동 — AI 에이전트가 DBTower를 도구로 씁니다

        
앱이 떠 있으면 준비 완료. 아래 명령 한 줄이면 Claude Code가 시점 비교·실행계획 분석을 직접 수행합니다.

        

          claude mcp add --transport http dbtower http://localhost:8080/mcp   (ADMIN 로그인 시 토큰 포함 명령으로 완성됩니다)
          
        

        

          claude mcp add dbtower -- sh scripts/dbtower-mcp.sh   (stdio 방식 — 앱과 분리 실행)
